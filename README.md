# Babita Kumari – Resume Website

A simple, clean, and semantic **HTML resume website** showcasing personal details, skills, experience, projects, and education.  

---

##  Features

- Semantic HTML5 structure (`header`, `main`, `section`, `footer`)
- Clean and readable resume layout
- Tables for Experience and Education
---

##  Technologies Used

- **HTML5**

---

##  Setup Instructions

Follow these steps to run the project locally:

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/babita-resume.git
   cd babita-resume
   ```
2. Open with Live Server in VS Code
3. Open index.html in any modern web browser
